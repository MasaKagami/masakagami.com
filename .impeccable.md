## Design Context

### Users
Visitors to Masa Kagami's personal portfolio: prospective clients, recruiters, fellow designers/developers, and collaborators evaluating his work in product development, UI/UX, and frontend engineering. They arrive with curiosity, scanning for craft, taste, and signal. The job to be done is to quickly read who Masa is, see proof of work, and (when convinced) reach out.

### Brand Personality
**Editorial · Restrained · Crafted.** The site reads like a print magazine: assured, quiet, and meticulously typeset. It evokes confidence and calm — never loud, never decorative for decoration's sake. Every element earns its place.

### Aesthetic Direction
- **Off-white canvas, near-black ink.** Background `#f3f3f3`, foreground `#1a1a1a`. No accent color — contrast is the only color hierarchy.
- **Sharp corners.** No rounded surfaces beyond circular controls. Borders are thin (1px, `#d4d4d4`).
- **Editorial typography.** Poppins for nearly everything (UI, headings, body). Lora reserved for the wordmark only. Big display sizes use tight tracking and tight leading.
- **Uppercase muted eyebrows.** Section labels, field labels, and metadata use `text-xs uppercase tracking-wider` in `--text-muted`. This is a recurring rhythm — use it.
- **Lots of breathing room.** Generous vertical spacing between sections (`pt-20 md:pt-32`). Resist the urge to compact.
- **Motion is cinematic, not bouncy.** Easing is `cubic-bezier(0.16, 1, 0.3, 1)` (refined ease-out). Reveals combine fade + translateY + brief blur. Stagger delays create choreography, not chaos. Never use bounce/elastic curves.
- **Anti-references**: SaaS dashboard chrome, gradient buttons, drop shadows used as decoration, rounded "friendly" UI, vibrant accent colors, cluttered cards.

### Design Principles
1. **Earn every pixel.** If a divider, icon, or label doesn't increase clarity or rhythm, remove it.
2. **Hierarchy through scale and weight, not color.** Black-on-off-white only — let typography do the work.
3. **Eyebrow / title / body.** New surfaces follow the same three-part rhythm: tiny uppercase muted label → confident title → calm supporting text.
4. **Motion serves comprehension.** Animations explain spatial relationships and acknowledge interaction. They never call attention to themselves.
5. **Respect reduced motion.** Every animation has a `prefers-reduced-motion` fallback. This is not optional.
